# Epsilon-NFA-to-DFA
Place the ```input.txt``` file in ```Tests/Inputs```   
Run program   
Output will be in ```Tests/Output```   
Input.txt format:
```text
a b c  //alphabet
P Q R  //states
P      //init state
R      //final states
P c R  // transitions
P λ R
P λ Q
P b Q
Q a P
Q c P
Q c Q
Q b R
```
